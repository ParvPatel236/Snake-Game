# Snake-Game
A simple Snake Game built in C++ using Raylib. The player controls the snake to eat food and grow longer while avoiding collisions. A fun project to practice game loops, input handling, and basic graphics with Raylib.

# Raylib C++ Starter Template

Raylib C++ Starter Template for Visual Studio Code on Windows.

This demo project contains a bouncing ball raylib example program.

It works with raylib version 4.5.
Tested on both Windows 10 and Windows 11.
